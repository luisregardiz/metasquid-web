## Getting Started

### Prerequisites
You need to have Node.js installed
* [Node.js](https://nodejs.org/es/)


### Installation

1. Clone the repo
   ```sh
   https://github.com/luisregardiz/metasquid-web.git
   ```
2. Install  packages
   ```sh
   npm install
   ```
3. Run dev server
   ```sh
   npm run dev
   ```

4. Run the production build 
   ```sh
   npm run build
   ```
